HADES is an IoT-based household safety monitoring system designed to detect potentially dangerous unattended heat sources by combining infrared temperature sensing, mmWave human presence detection, local alarm hardware, and cloud-based communication through AWS.

About the Project

HADES is a smart household safety monitoring system built around the CC3200 LaunchPad. The purpose of the device is to detect unsafe situations where a strong heat source is present while no person is nearby to monitor it.

To do this, the system reads temperature data from an IR sensor and presence information from a mmWave radar module. Based on those readings, the controller decides whether the situation is safe, whether a warning should be sent through AWS, or whether a local alarm should be activated until the user acknowledges the event.

System Architecture

The system architecture centers around the CC3200 board, which interfaces with multiple peripherals and sensors. The IR sensor communicates over I2C, the mmWave radar communicates over UART, and the Adafruit OLED display is controlled over SPI. A GPIO output is used to control the relay and alarm path, while a GPIO input is connected to a user button for acknowledgement or control actions.

Power is distributed through the system using 5V and 3.3V connections as needed for the board and peripheral devices.

System State Machine

The HADES controller operates using a state-based workflow. After initialization, the system attempts an AWS handshake and then continuously polls for presence and heat conditions. If heat is detected while no human presence is found, a timer begins.

Once the unattended heat condition persists long enough, the system attempts to notify the user through AWS. If acknowledgement is not received within the allowed time, a local alarm is activated until the button is pressed to acknowledge the warning.

Project Goals

The primary goal of HADES is to improve home safety by detecting hazardous unattended heat sources before they become more serious problems. By combining heat sensing with occupancy awareness, the system can reduce false alarms while still reacting quickly to potentially dangerous situations.

This project demonstrates embedded system integration across sensing, communication, state-machine control, user notification, and local hardware response.

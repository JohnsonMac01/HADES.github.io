About the Project

HADES is an IoT-based household safety monitoring system designed to reduce fire hazards caused by unattended heat sources. The system combines an MLX90614 non-contact infrared temperature sensor with a human presence sensor to determine when a dangerous condition may exist.

If a significant heat source is detected while no human presence is sensed for a period of time, the device can alert the user through a cloud-based notification system.

System Architecture

The system uses a CC3200 LaunchPad as the main controller. Sensor data is collected and processed locally, displayed on an OLED screen, and sent to the cloud when an alert condition is detected.

Project Goals and Results

The goal of this prototype is to detect potentially dangerous unattended heat sources and provide a real-time warning to the user. By combining environmental sensing with presence detection, the system improves household awareness and safety.

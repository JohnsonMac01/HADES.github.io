# Project Description
HADES is an IoT safety monitoring system designed to detect unattended heat sources in the home. The system combines a non-contact infrared temperature sensor with a mmWave human presence radar.
If a high temperature source is detected while no human presence is sensed, the system sends a real-time alert to the user through AWS IoT.
